# tanyAIn  – Teman Belajar Interaktif Kamu

**tanyAIn** adalah asisten belajar pintar berbasis AI yang dirancang khusus untuk siswa. Bukan cuma sekadar tanya-jawab, tanyAIn hadir dengan fitur latihan soal interaktif dan manajemen tugas (To-Do List) agar belajar jadi lebih teratur dan menyenangkan.

---

## ✨ Fitur Utama

### 1. 🧠 Tanya AI (Chatbot)
Tanya apa saja, dari rumus Matematika sampai sejarah dunia. tanyAIn akan menjawab dengan bahasa yang simpel, santai, dan mudah dimengerti (seperti ngobrol dengan kakak kelas!).

### 2. 📝 Latihan Soal Interaktif
Uji pemahamanmu dengan kuis yang menantang:
* **Mode Kuis:** Latihan soal pilihan ganda atau isian singkat.
* **Feedback Instan:** tanyAIn bakal kasih tahu jawabanmu benar atau salah beserta penjelasannya.
* **Level Progresif:** Soal akan menyesuaikan dengan tingkat kemampuanmu.

### 3. ✅ To-Do List Siswa (Smart Task)
Kelola tugas sekolahmu tanpa ribet:
* **Input Tugas:** Catat PR, proyek kelompok, atau jadwal ujian.
* **Prioritas Otomatis:** tanyAIn bantu urutkan mana yang harus dikerjakan duluan.
* **Interaktif:** Tandai tugas yang sudah selesai dan dapatkan apresiasi kecil dari tanyAIn!

## 🚀 Cara Memulai

### Prasyarat
* Node.js
* API Key Gemini

### Instalasi
1. Clone repositori ini:
   git clone [https://github.com/RyyzDev/tanyain-ai-chatbot.git](https://github.com/RyyzDev/tanyain-ai-chatbot.git)

2. Masuk ke direktori proyek:
   cd tanyain-ai-chatbot

3. Install dependensi:
   npm install

4. Jalankan aplikasi:
   npm start
